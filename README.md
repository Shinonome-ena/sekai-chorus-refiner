# 🏆 排行榜截图处理工具

自动裁切排行榜截图中的圆角矩形卡片，去除"你"标签，输出透明背景 PNG。

> 适用于音乐/游戏类 App 排行榜截图的批量处理，支持单张和批量操作。

---

## ✨ 功能

- **裁切圆角矩形** — 自动识别截图中的卡片区域，裁切并生成圆角蒙版
- **去除"你"标签** — 自动检测并覆盖排行榜中高亮的"你"条目
- **首末行裁切** — 可选去除卡片首尾多余的行
- **批量处理** — 支持整个文件夹一键处理
- **GUI / 命令行** — 提供图形界面和命令行两种使用方式
- **LSB 溯源标记** — 输出图片自动嵌入作者信息，可通过验证脚本溯源

---

## 📦 安装

```bash
pip install -r requirements.txt
```

依赖：`opencv-python` `numpy` `Pillow` `customtkinter`

---

## 🚀 使用

### GUI 版（推荐）

```bash
python app.py
```

1. 选择输入文件夹或图片文件
2. 选择输出文件夹
3. 选择处理模式
4. 点击「开始处理」

### 命令行版

```bash
python run.py
```

交互式菜单操作：

| 选项 | 功能 |
|------|------|
| 1 | 选择图片目录 |
| 2 | 一键全流程（留有首末行） |
| 3 | 一键全流程（不留首末行） |
| 4 | 仅裁切圆角矩形 |
| 5 | 仅去除"你"标签 |
| 6 | 仅首末行裁切 |

## 🔍 溯源验证

所有通过本工具输出的图片均自动嵌入 LSB 隐写标记（肉眼不可见）。

```bash
# 验证单张图片
python verify_stego.py image.png

# 验证整个文件夹
python verify_stego.py output/
```

检测到标记时会显示作者信息，可用于证明图片由本工具生成。
